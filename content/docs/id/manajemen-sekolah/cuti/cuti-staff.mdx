## Kelola Pengajuan dan Riwayat Cuti Seluruh Staff Sekolah dengan Praktis dan Transparan

Menu **Cuti Staff** dirancang khusus untuk mendukung admin sekolah dalam memantau, menyetujui, dan mencatat riwayat cuti para tenaga pendidik dan staf lainnya secara terpusat. Proses pengajuan hingga verifikasi cuti kini dapat dilakukan langsung melalui aplikasi mobile eSchool, kapan saja dan di mana saja.

# Fitur Utama

1. **Pengajuan Cuti Digital**

        Staff dapat mengajukan permohonan cuti melalui sistem dengan input tanggal, jenis cuti, serta alasan. Semua data langsung tersimpan dan diteruskan ke admin terkait untuk proses persetujuan.

2. **Filter Berdasarkan Tahun & Bulan**

        Admin dapat menyaring pengajuan cuti berdasarkan tahun ajaran dan bulan tertentu sehingga proses pencarian dan peninjauan data lebih efisien. Cocok untuk menyusun rekap tahunan atau bulanan.

3. **Notifikasi & Status Real-Time**

        Setiap pengajuan akan menampilkan status:

        * **Tertunda** – Menunggu persetujuan
        * **Disetujui** – Telah disetujui oleh admin
        * **Ditolak** – Pengajuan tidak dikabulkan

        Status ini membantu staff memantau progres permohonan cuti secara langsung.

4. **Riwayat Cuti Terintegrasi**

        Seluruh riwayat cuti (baik aktif maupun tidak aktif) tercatat secara otomatis. Admin dapat memeriksa frekuensi, jenis cuti, serta distribusinya selama periode berjalan.

5. **Transparansi dan Akuntabilitas**

        Setiap keputusan cuti dapat dilacak dan diaudit, sehingga menjaga transparansi antara staf dan manajemen. Tidak ada lagi pengajuan manual yang tercecer atau tidak terdokumentasi.
