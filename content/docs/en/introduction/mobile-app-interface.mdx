# Main Functions in Mobile Dashboard View

The **Mobile Dashboard** page presents real-time data summaries that are directly relevant to users, such as:

* Displays the number of **teachers**, **students**, and school staff
* List of **classes and majors**
* List of today's class schedules
* Recent activity notifications (announcements, assignments, attendance)

With this view, **teachers and school staff** can monitor school dynamics practically and instantly, without having to open a laptop or manual system.

This view facilitates the school in monitoring current activities and supports quick and accurate decision-making.

<table><thead><tr><th width="133">Main Function</th><th>Function</th></tr></thead><tbody><tr><td>School Announcements</td><td>Displays a list of important announcements chronologically</td></tr><tr><td>Active Academic Year</td><td>Displays the currently running academic year</td></tr><tr><td>Quick Statistical Data</td><td>Provides an instant summary of the number of teachers, active/inactive students, guardians, etc</td></tr><tr><td>Class Schedule List</td><td>Displays all class schedules taught by the teacher today</td></tr></tbody></table>
