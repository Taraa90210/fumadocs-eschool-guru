## Manage and Conduct Digital Exams Practically and Structuredly

The **Online Exam** menu is the control center for creating, managing, and conducting digital-based exams. This feature provides high flexibility for educators in compiling questions, setting implementation times, to monitoring student exam implementation in real-time.

# Features:

1. **Exam Filter**

    Use dynamic filters to screen the exam list based on:

    * **Class**
    * **Subject**

    This feature facilitates searching for relevant exams, especially when handling many classes or exam topics at once.

2. **List of Created Exams**

    Each exam entry displays important information briefly, including:

    * **Exam Title**
    * **Start and end time**
    * **Exam duration (in minutes)**
    * Quick access to **question settings** and **archive/edit** features

With a neat visual design, teachers can monitor exam status and make changes quickly in just a few taps.
