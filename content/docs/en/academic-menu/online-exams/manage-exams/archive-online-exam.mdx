## Manage and Track Your Entire Digital Exam History Easily

The **Archive Online Exam** menu is designed to provide fast and organized access to all completed or inactive exams. Teachers can save, search, and restore archived exam data, as well as maintain order in long-term exam management.

# Key Features

1. **Archive Exam Search**

        Search for specific exams from the archive list easily using keywords for exam names, subjects, or dates. This feature is very useful for tracking old exams that you want to access again.

2. **Archive Date Filter**

        Set a date range to display only exams that fall within a certain time period. This helps speed up searches when handling large amounts of exam data.

3. **Manage Archives Flexibly**

        For each archived exam, users can:

        * **Restore Exam** — Reactivate archived exams to be edited or reused.
        * **Permanently Delete** — Completely remove exams from the system if they are no longer needed.

4. **Saved Exam Information**

        Each exam displayed in the archive includes:

        * **Exam Title**
        * **Subject & Category**
        * **Implementation Date & Time**
        * **Exam Duration (in minutes)**
        * **Status Label: Archive**

With a neat and informative display, teachers can review all important details without having to open each item manually.

# Benefits of Archive

Saving exams to the archive not only helps efficiency, but also maintains data integrity and readiness if at any time it needs to be reused for reference, evaluation, or material adjustments.
