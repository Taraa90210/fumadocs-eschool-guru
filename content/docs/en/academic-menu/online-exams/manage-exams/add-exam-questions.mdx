## Select, Browse, and Insert Questions Quickly from Integrated Question Bank

The **Add Online Exam Questions** menu provides ease for educators in compiling exam questions quickly and flexibly. With a dynamic and accessible interface design, you can directly select questions from the available question bank without needing to re-create them from scratch.

# Steps to Add Questions

1. **Enter Question List**

        View the list of all questions that have been added to the exam. You can monitor the number of questions, question types, points, and versions in detail in one view.

2. **Select Question Bank**

        Access various question bank categories that you have, such as:

        * Essay Questions
        * Multiple Choice
        * True/False
        * Short Answer
        * Numeric

        Each category displays the number of available questions, making it easier for you to choose relevant banks.

3. **Browse & Select Questions**

        Enter the selected question bank and:

        * Use the search feature to find specific questions.
        * Press directly on the question to select and view its full content.
        * Slide to the side to browse **previous question versions** and compare content.
        * Check point weights and question types before adding.

4. **Confirm & Save Questions**

        After questions are selected:

        * Press the **“Save X Questions”** button to insert questions into the online exam.
        * You can add several types of questions in one exam to increase assessment variety.

This feature is designed to make it easier for teachers to compile exams in an efficient, structured, and richly varied way—presenting a fair and comprehensive assessment experience for students.
