## Monitor Student Exam Grades Directly, Practically, and Transparently

The **Online Exam Results** menu is designed to help teachers monitor, correct, and recap all online-based exam results efficiently. All data is presented in real-time in a clean, modern, and intuitive visual display — from exam names to total completion duration.

# Key Features

1. **Structured Exam Summary**

        Each exam is displayed in a card format containing:

        * Exam title and subject
        * Implementation date and duration
        * Status indicator (✅ Finished / ⏳ In Progress)

        Facilitates users in selecting and checking exam results based on time and type.

2. **View Result Details**

        Press the **“View Exam Results”** button to access:

        * Grade recap for each student
        * Question performance (for multiple choice and essays)
        * Easy-to-read student response display

        All these components support in-depth evaluation of student achievement in one place.

3. **Manual Essay Grading**

        For essay questions:

        * Teachers can view complete student answers
        * Input scores manually (0–100) per question
        * Direct grading from the same screen

        Very useful for providing rubric-based assessments or direct observations.

4. **Validation & Automatic Storage**

        After all grades are checked:

        * The system automatically saves the results
        * Grades are archived and ready to be sent to the central dashboard or homeroom teacher

        Ensures no data is missed or lost.
