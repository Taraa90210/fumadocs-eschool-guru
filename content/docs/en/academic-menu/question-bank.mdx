The **Question Bank** menu is a main feature designed to help educators manage, compile, and store various types of questions systematically in one place. With an intuitive interface and professional design, this feature makes it easy to create and group questions according to learning needs, whether for independent practice, quizzes, or school exams.

# **Supported Question Types :**

To answer diverse evaluation needs, the Question Bank system provides **5 question types** that can be selected when creating a new question:

1. **Multiple Choice**\
   Questions with several answer options, where only one is correct. Equipped with correct answer marking features and specific feedback per option.
2. **Essay (Long Description)**\
   Used to measure students' analytical, reasoning, and writing skills. Answers are open-ended and graded manually by the teacher.
3. **True / False**\
   Questions in the form of statements, where students only choose between two options: _True_ or _False_. Suitable for quickly testing understanding of basic concepts.
4. **Short Answer**\
   Assess student understanding through responses in the form of short words or phrases. Answers can be graded automatically if single or multiple answer keys have been set.
5. **Numeric**\
   Questions where the answer is a number or numeric value. Ideal for subjects like Mathematics, Physics, Accounting, or Economics

# **Question Management Features**

1. **Question Bank Categorization**\
   Each question bank can be created based on topic, subject, chapter, or exam type. This facilitates grouping and searching for questions.
2. **Flexible Question Creation**\
   The question creation form provides complete fields: question name, main question, additional notes, point settings, and question types that can be selected as needed.
3. **Supporting Material (optional)**\
   Teachers can add files or multimedia references to enrich the question context.
4. **Edit & Duplicate Questions**\
   Saved questions can be easily updated or duplicated for reuse in different exam sessions.
5. **Automatic & Manual Grading**\
   For objective questions like multiple choice, true/false, and numeric, grading is done automatically. While essays and short answers can be graded manually for more flexibility.

# **Benefits of Question Bank for Teachers & Educational Institutions**

1. Accelerate the process of creating exams and student exercises.
2. Ensure consistency and quality of questions.
3. Increase efficiency in carrying out evaluations.
4. Provide a systematic track record and documentation of questions.
5. Can be used across semesters, across subjects, and across levels.
